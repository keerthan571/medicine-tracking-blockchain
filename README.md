# Blockchain-Based Medicine Tracking System

## 📌 Project Overview

A system to track medicines using blockchain to ensure authenticity and prevent counterfeit drugs.

## 🧩 Modules

* Frontend (UI)
* Backend (API)
* Blockchain (Smart Contracts)

## ⚙️ Tech Stack

* Frontend: HTML/CSS/JS or React
* Backend: Node.js + Express
* Blockchain: Ethereum (Hardhat)

## 👥 Team Division

* Member 1: Authentication
* Member 2: Frontend
* Member 3: Blockchain (Smart Contracts + API)
* Member 4: Integration

## 🚀 Workflow

Frontend → Backend → Blockchain
